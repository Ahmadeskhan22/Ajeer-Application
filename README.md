As young graduates in Jordan, one of the biggest challenges we observe in our local community is youth unemployment. After extensive research, we realized there is a major gap in the market: students searching for flexible work to support their studies, and skilled workers being sidelined simply because they lack academic credentials. Most existing solutions are scattered across chaotic channels, with unclear wages and shift duties.
To solve this, we designed, built, and successfully presented: "AJEER |  A Smart, Flexible Work Matching Platform.
AJEER is a synchronized web and mobile ecosystem built to streamline part-time and temporary hiring. It bypasses academic gatekeeping and connects people directly to opportunities based strictly on their skills, geographical location, and precise hourly availability.
To pull off a project of this scale, our team divided and conquered the software engineering pipeline into dedicated specialized roles:
 My Role:                                                                                                           Mobile Application Developer I engineered the cross-platform mobile application dedicated exclusively to job seekers using Flutter & Dart. I focused heavily on building a reactive user experience, integrating local state management with Provider, handling secure session caching, and managing real-time data synchronization with our backend APIs.
 Abdulrahman Saleh: Systems Analyst & Frontend Web Developer He spearheaded the architectural analysis, data gathering, database structure design, and developed the responsive web front-ends utilizing Bootstrap, ensuring a flawless presentation for web-based job seekers.
 Ahmad Alhemsi: Backend Developer & Core Systems Engineer He engineered the centralized server infrastructure using PHP & Laravel 10, secured API endpoints with Laravel Sanctum, built the complex administrative moderation dashboard, and implemented our core multi-criteria matching scoring algorithm.
 Our Unified Engineering Stack:
Mobile: Flutter & Dart | Provider State Management | GoRouter
Backend & Web: PHP 8.2 | Laravel 10 | Blade Templates | Bootstrap 5 | MySQL


Security & DevTools: Laravel Sanctum | CSRF & Bcrypt Protection | Git & GitHub
Methodology: Agile SDLC (Iterative Sprints)
